## 2. Current State Assessment

### 2.1 Widget Creation Flow

| Step | Current Method | SME Friendly? |
|------|---------------|---------------|
| 1. Create widget | Settings → Widget → Manual form | ❌ Technical |
| 2. Upload documents | Separate Documents page | ❌ Disconnected |
| 3. Configure theme | Manual color picker | ⚠️ OK |
| 4. Set welcome message | Text input | ⚠️ OK |
| 5. Get embed code | Copy from widget settings | ⚠️ Hidden |
| 6. Deploy | Manual paste into website | ❌ Technical |

### 2.3 User Journey Analysis

**Current Flow (6 clicks, 3 pages):**
1. Dashboard → Documents → Upload → Wait
2. Dashboard → Settings → Widget → Create
3. Configure theme → Save
4. Copy embed code
5. Paste into website

**Target Flow (1 page, 5 minutes):**
1. Dashboard → "Create Chatbot" wizard
2. Step 1: Upload documents
3. Step 2: Bot name
4. Step 3: Brand color
5. Step 4: Welcome message
6. Step 5: Preview
7. Step 6: Copy script

## 3. Gap Analysis

| Gap | Severity | Impact |
|-----|----------|--------|
| No wizard page | HIGH | SME users can't create chatbot easily |
| Disconnected upload flow | HIGH | Users must navigate multiple pages |
| No live preview | MEDIUM | Users can't see result before deploying |
| No guided setup | HIGH | Non-technical users get lost |
| No copy-paste output | MEDIUM | Embed code is hidden in settings |

---

## 4. Implementation Plan

### Phase 1: Wizard Page
1. Create `/wizard` route
2. 6-step wizard component
3. Step 1: Document upload (drag & drop)
4. Step 2: Bot name input
5. Step 3: Brand color picker
6. Step 4: Welcome message textarea
7. Step 5: Live preview (iframe)
8. Step 6: Generate & copy script

### Phase 2: Wizard API
1. Create `/api/wizard/create` endpoint
2. Combines: create widget + upload documents
3. Returns: widget config + embed code

### Phase 3: Testing
1. Playwright E2E test
2. Mobile responsiveness
3. Accessibility check

---

## 5. Recommendation

**GO** for implementation. The wizard will significantly reduce friction for SME users by combining 3 pages into 1 guided flow.
