## Note on Raw SQL

The `embedding` field is `Unsupported("vector(1536)")` in Prisma, which means INSERT/UPDATE operations for embedding MUST use raw SQL. This is documented in the code:

```typescript
// Note: embedding is Unsupported("vector") so we use $executeRaw for the INSERT
```

This is a Prisma limitation with PostgreSQL vector types, not dead code.
