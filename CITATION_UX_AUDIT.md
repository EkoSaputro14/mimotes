## 3. Gap Analysis

### 3.1 Widget Citation UX Gaps

| Gap | Severity | Impact |
|-----|----------|--------|
| No source cards | HIGH | Users can't identify source documents |
| No similarity badge | MEDIUM | Users can't gauge answer confidence |
| No expand preview | MEDIUM | Users can't verify content without clicking |
| No file type icons | LOW | Missing visual hierarchy |
| No inline citations | HIGH | Users can't trace claims to sources |
| No mobile optimization | MEDIUM | Poor experience on phones |

### 3.2 Requirements for SME Users

1. **Source Cards** — Visual cards showing document name, type, similarity
2. **Similarity Badge** — Color-coded confidence indicator (high/medium/low)
3. **Expand Source Preview** — Click to see content snippet
4. **Mobile Friendly** — 44px touch targets, responsive layout
5. **Accessible** — ARIA labels, keyboard navigation
